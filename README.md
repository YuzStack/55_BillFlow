BillFlow is a sophisticated subscription management application designed to help users track, visualize, and manage their recurring expenses. By centralizing subscription data and providing visual financial insights, BillFlow empowers users to take full control of their monthly spending.

## üß† The Design-to-Code Workflow

Unlike course-led projects, BillFlow was built using an original design pipeline. I leveraged Magic Patterns to generate the initial UI concept and then refined the design in Figma. This process eliminated styling guesswork and allowed me to focus entirely on high-fidelity implementation, ensuring the final code perfectly matched the intended visual architecture.

## ‚ú® Key Features

- **Subscription Management:** A full CRUD interface to add, edit, and track various subscription services and their costs.
- **Donut Chart Visualization:** Implemented a dynamic donut chart to provide a clear, high-level breakdown of expense distribution across different categories.
- **Real-time Financial Tracking:** Automatically calculates total monthly and yearly spends based on active user data.
- **Precision Styling with Tailwind:** Utilized Tailwind CSS to translate Figma designs into a responsive, modern interface with a focus on clean typography and spacing.
- **State-Driven Interactivity:** Managed complex application states to ensure seamless transitions between viewing list data and adding new entries.
